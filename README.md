
# WCM (Warvand Change Modules)

This project is a rewrite of the Mount & Blade: Warband non-working module replacement panel.


## Features

- Possibility to change modules
- Possibility to change keys (Native,Napoleonic Wars)
- Fast

  
## How to operate

```bash
python3 wcm-*.py

```
